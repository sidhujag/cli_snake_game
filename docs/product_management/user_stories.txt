User Stories:

1. As a player who has limited time, I want to be able to quickly start a new game with a single command, so that I can maximize my enjoyment without a lengthy setup process.

2. As a player seeking challenge, I want the game to progressively increase in difficulty as my score gets higher, so that I can be continuously challenged and engaged.

3. As a player who enjoys social gaming, I want to be able to share my high score on social media, so that I can compete with friends and show off my achievements.

4. As a visually impaired user, I want the game to have audio cues that help me understand the game state, so that I can enjoy playing without relying on the visual interface.

5. As a player who enjoys customization, I want to be able to change the appearance of the snake and the game board, so that I can personalize my gaming experience.
